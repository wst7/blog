我们运行转换后的代码，看一下Cat.proto

发现Cat.proto是指向Animal的，在转换后的代码中也能找到对应的地方，就是在_inherits函数最后一句

所以es6 class的继承实现就是在寄生组合式继承的基础上将子类的proto属性指向父类，为什么要这样做？
